rdfs:comment
The class of proposals. A distinction is made between a proposal and a document or set of documents that describe the proposal. Instances of the class :Proposal correspond to the action of making or submitting a proposal. The wdrs:describedby property may be used to relate a proposal to one or more documents that describe the proposal that is made.
